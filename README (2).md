# بوابة الفحص الصوتي

تطبيق ويب للتعرف الصوتي على أرقام وحروف لوحات السيارات باللهجة المصرية.

## طريقة الرفع والتشغيل على GitHub Pages

1. اعملي حساب على GitHub (لو معندكيش) من https://github.com
2. اعملي Repository جديد (New repository):
   - اكتبي اسم زي: `plate-scanner`
   - خليه Public
   - اضغطي Create repository
3. جوه الـ Repository، اضغطي **Add file → Upload files**
4. اسحبي ملف `index.html` المرفق هنا وارفعيه
5. اضغطي **Commit changes**
6. روحي على **Settings → Pages** (في القائمة الجانبية)
7. تحت **Build and deployment → Source** اختاري **Deploy from a branch**
8. اختاري Branch: `main` والفولدر `/ (root)` واضغطي **Save**
9. استني دقيقة، هيظهرلك رابط شكله:
   `https://USERNAME.github.io/plate-scanner/`
10. افتحي الرابط ده من **كروم** (على الموبايل أو الكمبيوتر) — المايك هيشتغل عادي لأن الرابط https آمن

## ملاحظة
لازم تفتحيها من الرابط ده (https)، مش بملف مباشر من الجهاز، عشان إذن المايك يشتغل.
